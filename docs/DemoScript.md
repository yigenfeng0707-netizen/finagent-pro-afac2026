## 场景一：智能体协同分析（60 秒）

> 我输入一只股票，6个专业智能体自动协同分析——市场分析看技术指标，新闻舆情抓情感，风控合规跑规则引擎，最后协商出投资建议。整个思维链实时可见。

3. **点击「开始分析」**
4. **观察思维链可视化**（ThinkingChain 组件）：

   - 📰 新闻舆情智能体：近期正面新闻占比72%

5. **查看综合信号徽章**：🟢 买入信号，置信度82%

### 录屏要点

- 重点录思维链逐步展开的过程，展示每个智能体的推理步骤
- 信号徽章从灰色变为绿色的瞬间

- 「6个智能体协同，不是串行等待，而是并行推理」

## 录屏操作清单

### 录屏前准备

- [ ] 后端启动：`cd backend && python -m uvicorn main:app --port 8000`
- [ ] 前端启动：`cd frontend && npm run dev`
- [ ] 浏览器全屏，隐藏无关标签页和书签栏
- [ ] 关闭其他应用通知（微信、邮件等）
- [ ] 录屏软件就绪（推荐OBS，1920x1080，30fps）

### 录屏技巧

1. **先录操作，后配音**：先录屏操作画面，后期用剪辑软件配音，避免紧张忘词
2. **鼠标移动要慢**：让评委看清每一步操作
3. **关键画面停留2-3秒**：思维链展开、信号徽章变色、预警弹出
4. **不要录到终端窗口**：只录浏览器画面
5. **如果API响应慢**：可以提前触发一次分析让数据缓存，录屏时响应更快

### 推荐录屏流程

1. 打开Dashboard → 停留3秒展示整体布局
2. 切换Analyze → 输入600519 → 点击分析 → 等待思维链展开
3. 切换Chat → 输入问题 → 等待流式输出 → 输入追问
4. 切回Dashboard → 展示预警面板
5. 切换Alerts → 展示预警详情和合规标签
6. 切回首页 → 停留2秒
