# FinAgent Pro vs agentUniverse 对比

> AFAC2026 生态契合说明

## 理念契合

| 维度 | agentUniverse（蚂蚁开源） | FinAgent Pro |
|------|--------------------------|--------------|
| 核心理念 | 多智能体协同（PEER） | 6 智能体 + Orchestrator 协商 |
| 目标场景 | 金融通用 Agent 框架 | 智能财富投研垂直场景 |
| 协同模式 | 规划-执行-协同 | 三阶段流水线 + 加权协商 |

## 差异化优势

| 能力 | agentUniverse | FinAgent Pro |
|------|--------------|--------------|
| ReAct 推理 | 框架支持 | **自研引擎，10 步循环+滑动窗口** |
| 合规内嵌 | 需自行扩展 | **Day 1 规则引擎 + 审计中间件** |
| 主动智能 | 需自行实现 | **定时巡检 + 阈值预警 + WebSocket 推送** |
| 思维链可视化 | 无内置 UI | **ThinkingChain 组件实时展示** |
| 开箱产品 | 开发框架 | **完整 SaaS 产品（前后端+部署）** |

## 结论

FinAgent Pro 与 agentUniverse **理念同频、场景深耕**——不是重复造轮子，而是基于多智能体范式在 **智能财富** 方向的产品化落地。
