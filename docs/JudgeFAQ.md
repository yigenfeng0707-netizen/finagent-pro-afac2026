# FinAgent Pro · 评委 FAQ 手册

## Q1：和 ChatGPT / Copilot 有什么区别？

**A：** 通用 AI 是聊天工具，无金融专业工具、无合规审查、幻觉严重。FinAgent Pro 是 **6 专业智能体 + 工具调用 + 合规内嵌 + 思维链可视化**，定位是数字员工而非聊天机器人。

## Q2：和 Wind / 同花顺有什么区别？

**A：** 传统终端是 **被动数据查询**。我们是 **主动智能体**——7×24 巡检、预警推送、自动撰写研报，从工具到员工。

## Q3：如何避免 AI 幻觉？

**A：** 三重保障：① ReAct 推理每步绑定工具数据 ② 6 智能体交叉验证 ③ 置信度评分 + 合规审查。所有结论标注数据来源。

## Q4：合规怎么保证？

**A：** 风控合规智能体内嵌 4 条监管规则（单股/行业集中度、创业板/ST 限制），ComplianceAuditMiddleware 全链路留痕，每次建议自动过规则引擎。

## Q5：有没有落地案例？

**A：** 某高校金融科技实验室 14 天 POC：晨报撰写 ↓73%，预警响应从 4h → 3min，合规覆盖 100%。详见 `/cases` 页面。

## Q6：分析准确率如何？

**A：** Benchmark 脚本对 30 只 A 股样本评测，信号产出率、P95 时延、合规通过率均有量化报告（`benchmark/run_benchmark.py`）。

## Q7：数据安全？

**A：** 国产 LLM（DashScope/GLM/DeepSeek/SenseNova），数据不出境；支持私有部署；JWT 认证 + 审计日志。

## Q8：技术壁垒是什么？

**A：** ReAct 推理 + 三层记忆 + 多智能体协商 + 合规引擎，需金融+Agent 双领域经验，非简单 Prompt 工程可复制。

## Q9：商业模式是否可行？

**A：** SaaS 订阅 + 机构定制，Y1 目标 1200 万 ARR；个人版引流、机构版高 ARPU，LTV/CAC 目标 > 5:1。

## Q10：为什么选 agentUniverse 生态？

**A：** 理念一致（多智能体协同），我们自研 ReAct 引擎 + 合规内嵌 + 主动智能，更聚焦 **智能财富投研** 垂直场景。详见 `agentUniverseComparison.md`。
